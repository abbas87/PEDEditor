PED Viewer Help

Some boilerplate about finding the Editor at the NIST site.

File Menu

Save as PDF

Save as PNG

Save an image of the diagram. This can be done for all diagrams, including old ones for which PDF-format output was not previously available. Diagrams saved as PDF stay sharp at high magnifications, but diagrams saved as PNG do not.

Print

Print the diagram. The image will be scaled so that its margins just fit on the sheet.

Position Menu

In addition to moving the mouse pointer, the following commands are available for diagram positioning:

  • Select nearest point (Shift+Q) moves the mouse to the nearest point of special interest such as curve intersections, curve endpoints, and label and data point locations.
  • Select nearest line/curve (Shift+W) moves the mouse to the the nearest point on any ruler or curve.
  • Enter coordinates (Enter) allows you to type in the coordinates you would like the mouse to move to.

Holding the shift key down while moving the mouse will cause the mouse to be attracted to nearby curves and points of special interest.

Digitize Menu

    Files saved during export use the UTF-8 character encoding.

  • Export selected curve or label's coordinates: Export the coordinates of all curve control points and all labels to a file or to the clipboard. Coordinates

  • Export all curve and label coordinates: Export the coordinates of all curve control points and all labels to a file or to the clipboard. Coordinates belonging to a single curve and labels that share the same text are grouped together; groups are separated from each other by empty lines or comments.

    Checking the Add comments box will make the output easier to interpret but may make it harder to import the data into other programs. Comments begin with the sharp sign (#).

  • Press Control+Shift+S to copy the text in the gray status bar at the bottom of the edit window to the clipboard.

Chemistry Menu

  • Proportions/Convert to [weight/mole] fraction: Transform the diagram between weight and mole fraction values.

    This function works only for binary or ternary diagrams whose diagram components and extreme values have been entered correctly.

  • Formula to mole/weight fraction (also available by pressing the % key, that is (on US keyboards) Shift+5): Enter a chemical formula, and the mouse will move to the position in the diagram that that formula corresponds to.

View Menu

To zoom in, press the left button and drag the mouse until a red rectangle forms. Press Control+B (best fit) to exactly fill the edit window with the diagram.

Center mouse (Control+L) brings the part of the diagram under the mouse towards the center of the edit window. (The mouse may not end up in the center if that would bring empty space into view.)

Show slope window If checked, opens a window displaying the angle in screen coordinates and slope in principal coordinates (y/x for binary diagrams, top component over right compoentn for ternary diagrams) at the most recently selected location in the diagram.

Features Modified or Removed Since Version 3.4

  • Zoom out: Use Control+B or View/Best Fit.
  • Curve Follow: Replaced by Select nearest line/curve (Shift+W), though in some cases Select nearest point (Shift+Q) is more appropriate. You can use the arrow keys to move the mouse a small distance further along the curve and then press Shift+W again.
  • Lever Rule, Overlay, Reflection, and Rotate Axes: These features are no longer available with version 4.0.

  • Mole-to-Weight % and Weight-to-Mole %: For binary and ternary diagrams whose components have been entered as formulas and whose axes are set up correctly, the gray status bar at the bottom of the viewer displays both mole and weight percent at all times for the chosen location. Converting a diagram from one form to the other is a one-step operation. For other kinds of diagrams, these features are not available
