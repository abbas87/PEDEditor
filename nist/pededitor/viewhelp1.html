PED Viewer Help

Please note that this file will open using the default browser you have chosen for your computer, either full- or partial-screen, according to its last usage. If the last view was not full screen, the PED Figure Search interface will remain partially visible underneath.

The PED (Phase Equilibria Diagram) Viewer is a component of the PED Editor software package written by Eric Boesch to digitize phase diagrams for the PED Data Center. The full version of the PED Editor is available for downloading as freeware with some restrictions; see http://www.nist.gov/srd/nist31.cfm.

Some features may be accessed through the menu bar along the top edge of the Viewer screen, while others are accessed through a menu that is raised by pressing the right mouse button. These functions (and their keyboard shortcuts) are described below.

File

Open

Load a .PEDV file manually.

Many users will never need to use this option. However, some of you may have system configurations that prevent the PED Viewer from opening automatically when you click or download .PEDV files. Your IT specialist may be able to help you resolve such problems, but they should not prevent you from using the program.

Save as PDF

Save an image of the diagram. Diagrams saved as PDF stay sharp at higher magnification.

Save as PNG

Save as GIF

Save as JPEG

Save an image of the diagram. Diagrams saved in these formats do not stay sharp at higher magnification.

Print

Print the diagram. The image will be scaled so that its margins just fit on the sheet.

Exit

Closes the Viewer screen.

Exit all

Closes all Viewer screens at once.

Position

  • Enter coordinates (Enter) allows you to type in the coordinates you would like the mouse to move to.

Holding the shift key down while moving the mouse will cause the mouse to be attracted to nearby curves and points of special interest.

Digitize

Files saved during export use the UTF-8 character encoding.

  • Export selected curve or label's coordinates: You must already have selected a curve or label or data point. If you selected a curve, this will export the coordinates of all of its control points to a file or to the clipboard. If you selected a label or data point, this will export the coordinates of all similar labels or data points.

  • Export all curve and label coordinates: Export the coordinates of all curve control points and all labels to a file or to the clipboard. Coordinates belonging to a single curve and labels that share the same text are grouped together; groups are separated from each other by empty lines or comments.

    Checking the Include comments in output box will make the output easier to interpret but may make it harder to import the data into other programs. Comments begin with the sharp sign (#).

Chemistry

  • Proportions/Convert to [weight/mole] fraction: Transform the diagram between weight and mole fraction values.

    This function works only for binary or ternary diagrams whose diagram components and extreme values have been entered correctly (verify by hovering over each diagram component and checking the accuracy of the values in the gray status bar).

  • Formula to mole/weight fraction (also available by pressing the % key, that is (on US keyboards) Shift+5): Enter a chemical formula, and the mouse will move to the position in the diagram that that formula corresponds to.

  • Swap components exchanges the position of the left and right components in binary diagrams. For ternary diagrams, any pair of components may be swapped, but the diagram may look odd after swapping the top component. All three components can be reordered using successive swaps.

View

Zoom in: Press and hold the left mouse button and drag down and to the right to open a red string box. When the mouse is released the area in the box will be enlarged.

Best Fit (Control+B), after re-sizing the Viewer window (by dragging the lower right corner), will exactly fill the Viewer window with the diagram. This is also how you zoom out.

Show slope window: If checked, opens a window displaying the angle in screen coordinates and slope in principal coordinates (y/x for binary diagrams, top component over right component for ternary diagrams) at the most recently selected location in the diagram.

Popup Menu

To open the popup menu, press the right mouse button while the mouse is inside the diagram. If you hold down the Shift key as you do this, the operation will apply to the nearest point of special interest. Otherwise, the operation will apply to the current mouse position.

  • If you previously chose Select nearest line/curve or Select nearest key point, Deselect (Escape) unselects the item.
  • Jump Up, Down, Left, or Right (Shift+Up/Down/Left/Right arrow) moves the mouse to the closest curve in the given direction. For example, in binary temperature/composition diagrams, repeated use of Shift+Left and Shift+Right allows you to see each phase transition for a fixed temperature, while Shift+Up and Shift+Down allow you to see each phase transition for a fixed composition.
  • Step Up, Down, Left, or Right (Up/Down/Left/Right arrow) moves the mouse in small increments.
  • Select nearest key point (Shift+Q) moves the mouse to the nearest point of special interest such as curve intersections, curve endpoints, and label and data point locations.
  • Select nearest line/curve (Shift+W) moves the mouse to the nearest point on any ruler or curve.
  • Copy coordinates to clipboard copies the coordinate information in the gray status bar at the bottom of the Viewer screen to the clipboard.

Curve Follow

Replaced by Select nearest line/curve (Shift+W), though in some cases Select nearest key point (Shift+Q) is more appropriate. You can use the arrow keys to move the mouse a small distance further along the curve and then press Shift+W again.

Mole/Weight % Readout

For binary and ternary diagrams whose components have been entered as formulas and whose axes are set up correctly, the gray status bar at the bottom of the viewer displays both mole and weight percent (as mol%/wt%, even if the diagram units are wt%) at all times for the mouse position. Converting a diagram from one form to the other is a one-step operation using the Chemistry menu (Proportions). For other kinds of diagrams, these features are not available.

Lever Rule

This feature is no longer available with version 4.1. However, you can achieve a similar effect with the following procedure:

  1. Select a position in the diagram using the mouse or the Enter Coordinates command.
  2. Press Shift+Left Arrow to jump to the phase transition on the left, then write down the mole percent concentration given in the gray status bar at the bottom of the diagram.
  3. Press Shift+Right Arrow to jump to the phase transition on the right, then write down the given mole percent concentration.
  4. Use a calculator to divide the distance from the phase transition on the left to your original point by the distance between the left and right phase transitions.

Please send feedback to Phase3@ceramics.org.

Uninstalling the program

This program uses Java Web Start. To uninstall a Java Web Start program in Windows 7, use the following procedure. The details may vary depending on your operating system and the version of Java you are running.

  1. Open the Java Control Panel.

    Under Windows 7, you can locate the Java control panel by opening the (regular Windows) Control Panel, which is one of the options listed in the right side of the start menu, then entering Java in "Search Control Panel", and clicking on the result.

  2. From the "General" Tab, under "Temporary Internet Files", press the "View..." button. This should open the Java Cache Viewer.
  3. One of the applications listed should be the Phase Equilibria Diagram Viewer. Click on it and press the red X to delete.
