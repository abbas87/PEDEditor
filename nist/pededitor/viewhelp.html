PED Viewer Help

Note: This help file only describes the functions most likely to be useful to viewers of PED diagrams. For access to advanced features, select the Properties menu and check the box next to Show Editing Options. Afterwards, you can re-open the Help file to obtain a more complete description of these features as well as descriptions of additional features.

File Menu

Save as PDF

Save as PNG

Save an image of the diagram. Diagrams saved as PDF tend to be of higher quality than diagrams saved as PNG.

Print

Print the diagram. The image will be scaled so that its margins just fit on the sheet.

Position Menu

In addition to moving the mouse pointer, the following commands are available for diagram positioning:

  • Nearest point (q) moves the mouse to the nearest point of special interest such as curve and line intersections and endpoints and label and data point locations.
  • Nearest line/curve (w) moves the mouse to the the nearest point on any ruler or curve.
  • Enter coordinates (Enter) allows you to type in the coordinates you would like the mouse to move to.

Holding the shift key down while moving the mouse will cause the mouse to be attracted to nearby curves and points of special interest.

Digitize Menu

    Files saved during export use the UTF-8 character encoding.

  • Export all text: Export all labels, key values, tags, and diagram components, with duplicates removed. Each entry is terminated by a newline.

  • Export all curve and label coordinates: Export the coordinates of all curve control points and all labels. Coordinates belonging to a single curve and labels that share the same text are grouped together; groups are separated from each other by empty lines or comments.

    Checking the Add comments box will make the output easier to interpret but may make it harder to import the data into other programs. Comments begin with the sharp sign (#).

  • Press Control+Shift+S to copy the text in the gray status bar at the bottom of the edit window to the clipboard.

Chemistry Menu

  • Proportions/Convert to [weight/mole] fraction: Transform the diagram between weight and mole fraction values.

    There is no guarantee that every point on every curve of the diagram will be correct after you perform a conversion, and points outside the diagram may not line up very well at all. The diagram may look funny, especially if there is a wide divergence between the mole and weight percents.

  • Formula to mole/weight fraction (also available by pressing the % key, that is (on US keyboards) Shift+5): Enter a chemical formula, and the mouse will move to the position in the diagram that that formula corresponds to.

  • Copy all formulas to clipboard: Copy all unique formulas in the diagram to the clipboard in alphabetical order, one per line.

View Menu

To zoom in, press the left button and drag the mouse until a red rectangle forms. Press Control+B (best fit) to exactly fill the edit window with the diagram.

Alternatively, you can press '-' to zoom out and '+' to zoom in.

Center mouse (Control+L) brings the part of the diagram under the mouse towards the center of the edit window. (The mouse may not end up in the center if that would bring empty space into view.)
