Third-party library credits

In addition to core Java(TM) 1.7 libraries, this viewer/editor makes extensive use of open-source libraries. All credit for these libraries belongs with their respective authors.

  • The DejaVu LGC Sans and Serif fonts, version 2.33, with a few changes.
  • The JAMA Java matrix algebra package, version 20100222, for matrix reductions.
  • Jackson 1.9.3 for JSON file format support
  • iText 5.1.3 for PDF file format support
  • JSoup 1.6.3 for HTML-to-text conversion
Except for the DejaVu font, these libraries' source files remain unmodified. Libraries have been repackaged only to simplify compliance with Java JAR signing requirements.

Additional license information for these libraries, along with source code to this program, can be found by unzipping this JAR file and examining the contents of the licenses subdirectory and license notices on the libraries' respective source files.
