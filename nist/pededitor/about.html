Obtaining and using the source code

The source code to both the PED Editor and the PED Viewer is included in the JAR file for this product. You can use the jar utility, which is included in Oracle's Java Software Development Kit, to extract the contents of the JAR file for inspection. The contents of a JAR file can also be extracted using other utilities that handle ZIP files, because a JAR file is a kind of ZIP file.

Most of the PED Editor Java source code that does not belong to third-party libraries is in the public domain. Some source code requires direct or indirect use of the iText library, and the source and byte code for those files are subject to the AGPL. Comments at the top of each source code file indicates which is which. Programmers who do not wish to be subject to the AGPL may request a copy of the PED Editor that does not use iText or support PDF format.
